# Selenium Automation Framework – OpenCart

## Project Overview
This project is a Selenium-based UI automation framework designed to test key eCommerce user journeys on the OpenCart platform.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Extent Reports
- Maven

## Test Scenarios
1. Account Registration
2. User Login
3. Product Search
4. Add Product to Cart

## Framework Features
- Page Object Model (POM)
- Reusable test utilities
- Scalable test structure

## How to Run Tests

Clone the repo:

git clone https://github.com/Sithabilea/OpencartV121.git

Run using Maven:

mvn clean test
